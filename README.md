# AI-Based Aptitude Assessment Framework (AICA)

## Overview
This project proposes a new way to evaluate candidates in the AI era.

Traditional aptitude tests focus on logic and speed.
However, AI can now solve these instantly.

This framework shifts focus to:
- System design thinking
- AI guidance and prompting
- AI failure handling
- Ethical and security reasoning

## Key Contribution
- Designed AICA Framework (4 competency domains)
- Created 8 real-world assessment scenarios
- Developed structured evaluation rubric

## Why This Matters
Hiring systems are outdated relative to AI capabilities.
This project explores a more relevant evaluation model.
